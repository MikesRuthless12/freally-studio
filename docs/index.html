Freally Studio v0.20.02 — Initial Release

A full-featured Electron/React DAW with AI pattern generation across 35+ genres, an FL-style mixer, arrangement timeline, audio-clip editor, vocal pitch correction, real-time collaboration, and native WASAPI audio + VST3 plugin hosting.

What's Included

  • AI MIDI generators for chords, melody, bassline, and drums — all genre-aware across 35+ styles (Trap, Hip Hop, House, Techno, Drum & Bass, Jazz, and more).
  • FL Studio-style vertical mixer with per-channel faders, pan knobs, solo/mute, plugin racks, and a dedicated master bus.
  • Multi-track arrangement timeline — drag clips from generators, set loop regions, duplicate and splice clips.
  • Audio clip editor with trim, fade, normalize, reverse, 3-band EQ, and filter controls.
  • Built-in file explorer: import sample folders, preview audio, drag onto generators, and auto-detect key/BPM/genre via folder analysis.
  • Real-time P2P collaboration with PIN-authenticated rooms.
  • Export to WAV, MP3, and MIDI; full stem export for use in any external DAW.
  • Native WASAPI audio capture and VST3 plugin hosting in the Electron desktop build.

Freally Studio is completely free — every feature, for everyone. No Pro tier, no subscriptions, no payments, no ads.

Security & Correctness Audit Pass

Hardened Electron IPC and CSP, authenticated P2P collaboration with a PIN-protected mobile link, and robust MIDI/ZIP parsing — 844 tests passing, build clean.

What Changed

  • Electron desktop hardening — CSP, IPC allowlisting, shell.openExternal scheme validation, sender-frame checks across main, audioCaptureBridge, and vst3HostBridge.
  • P2P collaboration auth — crypto.getRandomValues IDs, URL-fragment room secret, host-signed privileged messages, schema-validated data channel.
  • Mobile-link PIN — 6-digit PIN delivered through QR fragment; TTL + idle expiry.
  • File parsing robustness — MIDI parser bounds, ZIP entry/size caps, manifest path allowlist, drag-drop magic-byte checks, SVG color sanitization.
  • Audio engine + lyrics fixes — tempo invariant clamp, drum solo-key consistency, AudioEngine timer dispose, rhyme-engine heteronym fix, caesura dash placement.
